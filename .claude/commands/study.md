Show the full CCA-F study plan for a specific day, week, or scenario.

Usage:
  /study day 4       → full plan for Day 4
  /study today       → calculate today's day from start date 2026-06-02, show that day's plan
  /study week 3      → show all 6 days in Week 3 with topics and objectives
  /study scenario 1  → show all days that build Scenario 1 (Customer Support Resolution Agent)

For /study day N or /study today, show:
  ══════════════════════════════════════════
  CCA-F Day [N]/36 — [topic]
  Week [W] · [domain] · [N]% complete
  ══════════════════════════════════════════
  🎯 Objective: [one sentence]
  📖 What to read: [bullet list with exact Skilljar module + docs link]
  💻 File: [weekN/wNdN_topic.py]
  ▶ Run: [exact command]
  🔢 Steps: [full numbered list — do not skip or shorten any step]
  🏗 Capstone layer: [exact file added to meterflow/]
  💬 Study questions: [3 questions]
  📝 Commit: git commit -m "[exact message]"
  ══════════════════════════════════════════
