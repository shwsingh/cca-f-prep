# MeterFlow SupportOps

CCA-F Certified Architect Foundations exam prep.
Building a production-grade multi-agent customer support system day by day across 36 days.

## What this is
Every day adds one real component to MeterFlow SupportOps — a billing support platform
that routes, resolves, and escalates customer tickets using Claude agents, MCP tools,
RAG, memory, and evaluation. By Day 36 it is a complete, runnable system covering
all 5 CCA-F exam domains.

## How to run
```bash
cd ~/Documents/cca-f-prep
source .venv/bin/activate
python week1/w1d1_setup.py
```
